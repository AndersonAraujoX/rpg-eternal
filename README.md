# Terras Eternas do Abismo

**Terras Eternas do Abismo** is an advanced idle RPG web game built with **React**, **TypeScript**, and **Vite**. It features a deep progression system, strategic combat, and automation mechanics, wrapped in a retro pixel-art aesthetic.

## üéÆ Game Features

### Core Mechanics
- **Eternal Battle Loop**: Fight endless waves of monsters and bosses that grow stronger with every defeat.
- **Hero Party System**: Recruit and manage a party of diverse heroes (Warrior, Mage, Healer, Rogue, Paladin, etc.).
- **Strategic Combat**: 
  - Elemental Affinities (Fire > Nature > Water).
  - Unique Class Skills and Ultimates.
  - Gambit System (AI scripting for automated combat decisions).

### Progression & Mastery (New!)
- **The Void Anvil**: Reforge your gear to roll perfect stats using Void Matter.
- **Achievement Mastery**: Unlock achievements to gain permanent global damage bonuses.
- **Combat Particles**: Visual feedback for damage, critical hits, and status effects.
- **DPS Meter**: Real-time tracking of your party's Damage Per Second.

### Deep Idle Strategy
- **Offline Progress**: Your heroes continue to fight and gather resources even when you're away.
- **Automation Constellations**: Unlock "Starlight" upgrades to automate equipment, tavern summons, and rebirths.
- **Infinite Tower**: Climb the Tower of Eternity for exclusive rewards and prestige.
- **Mining & Crafting**: Assign heroes to mine ore and forge powerful gear upgrades.

### Customization & Polish
- **UI Themes**: Choose from 5 themes (Default, Midnight, Forest, Crimson, Void).
- **Combat Log**: Detailed history of battle events.
- **Keyboard Shortcuts**: Hotkeys for quick navigation (S for Shop, I for Inventory, etc.).

## üöÄ Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AndersonAraujoX/rpg-eternal.git
   cd rpg-eternal
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
